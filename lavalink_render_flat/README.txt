# Lavalink Render Setup

This package is pre-configured for Render deployment.

## Steps

1. (Optional) Download Lavalink v4 manually:
   https://github.com/lavalink-devs/Lavalink/releases/latest

   Or let the start script auto-download it on first start.

2. Make sure start.sh is executable:
   chmod +x start.sh

3. Set Render start command to:
   ./start.sh

4. Default password: youshallnotpass
   Change it in application.yml for security.
